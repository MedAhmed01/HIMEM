# Aljazeera Font Files

Place the Aljazeera font files in this directory:

- `Aljazeera-Regular.woff2` - Regular weight
- `Aljazeera-Bold.woff2` - Bold weight

The font is configured in `app/layout.tsx` using `next/font/local`.

If the font files are not available, the system will fall back to Arial and sans-serif fonts.
